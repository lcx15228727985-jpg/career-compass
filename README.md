
# Career Compass V2 · 职业指南针

让用户通过「成就感经历」反推稳定优势与偏好，生成 30 个职业岗位推荐与职业路径可能性分析，帮助用户更早做对选择。

## 产品定位

**一句话描述**：通过「减法」做反向筛选，让人找到既钱又开心的职业方向。

**核心洞察**：选择比努力重要。多数工具解决的是「怎么找工作」，而 Career Compass 解决的是「怎么选方向」—— 不仅推荐岗位，还解释原因、分析风险、提供路径。

## 核心功能

### Step 1: 成就经历采集
- 引导式输入：至少 3 段「有成就感」的经历
- 每段包含：情境 → 行动 → 结果 → 感受
- 基于 NLP 推理用户稳定优势（能力簇）

### Step 2: 职业画像 + 偏好校准
- **职业画像（Career DNA）**：能力簇 + 驱动力 + 边界
- **30 个岗位分三层**：第一梯队（现在可试）/ 第二梯队（可迁移）/ 第三梯队（探索潜力）
- **薪资区间显示**：每个岗位显示薪资范围
- **用户编辑能力**：可编辑能力标签、驱动力、边界，系统自动重新计算匹配分
- **AI 教练对话**：系统追问验证用户偏好，3-5 轮对话完成校准

### Step 3: 职业模拟器
- 真实工作节奏模拟（5 个关键场景）
- 决策风格测试（4 道情境题）
- 输出：匹配分 + 优势信号 + 需关注卡点 + 现实预期

### Step 4: 职业发展路径
- 显示首选岗位的 1/3/5 年发展路径
- **双岗位对比图谱**：选择两个岗位生成知识图谱对比（15+ 节点），支持点击查看详情
- 支持切换主要展示的职业方向

### Step 5: 行动日历
- 30 天行动计划（按周分组：准备→验证→冲刺）
- 每个任务含标题 + 具体行动指引
- 支持标记完成状态
- 简历工作坊模块

## 技术栈

- **前端框架**：原生 JavaScript（ES6+）
- **构建工具**：Vite
- **样式**：CSS3（自定义属性、CSS 动画）
- **数据引擎**：mockEngine.js（本地模拟职业数据与推荐算法）

## 使用流程

1. **输入经历**：在 Step 1 添加至少 3 段成就经历
2. **生成分析**：点击「生成职业分析」进入 Step 2
3. **查看推荐**：浏览 30 个岗位推荐，可编辑画像、查看薪资
4. **完成校准**：与 AI 教练完成 3-5 轮对话验证偏好
5. **职业模拟**：在 Step 3 完成情境模拟测试
6. **查看路径**：Step 4 查看发展路径，可对比两个岗位
7. **行动指南**：Step 5 获取 30 天行动计划

## 核心算法

### 能力推理
- 基于正则规则从经历文本中提取能力信号
- 支持：数据分析、沟通协作、表达写作、用户洞察、执行推进、问题拆解、同理心、结构化表达

### 岗位匹配
- 计算用户能力簇与岗位标签的匹配度
- 结合发展性（growth）和压力（pressure）加权
- 考虑边界条件（高压/销售等）进行负向筛选

### 薪资模拟
- 基于岗位发展性生成模拟薪资区间
- 高发展性：14k ± 6k / 月
- 中发展性：10k ± 6k / 月
- 低发展性：8k ± 6k / 月
