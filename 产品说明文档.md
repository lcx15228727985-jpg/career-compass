# Career Compass 职业指南针｜产品说明文档

---

## 1. 产品定位

**Career Compass 职业指南针**是一款面向在校生、应届生和早期职场人（0-5年）的职业方向决策产品。

核心不是“投哪份工作”，而是**“先选对长期职业路径”**——帮助用户在“有成长、有收入、可持续”的轨道上长期发展。

产品通过用户过往**成就事件**与**偏好对话**，提炼职业画像（Career DNA），输出岗位推荐、双方案对比图谱和场景模拟体验，帮助用户做低成本但高质量的职业决策。

---

## 2. 目标用户

| 用户类型 | 核心痛点 |
|---------|---------|
| 在校生 / 应届生 | 经历碎片化，方向不清晰 |
| 0-3 年职场人 | 想转型但不确定适配方向 |
| 0-5 年早期职场人 | 需要在“收入、成长、稳定性”间平衡 |

---

## 3. 核心价值

- **先选路径，再做投递**：从 Career Development 出发，而不是仅做岗位匹配
- **可解释决策**：每个推荐与对比因子有理由，不是黑盒结论
- **先模拟再投入**：通过游戏化职业模拟器降低试错成本
- **人+AI混合咨询**：既可约真人前辈，也可即时使用前辈智能体

---

## 4. 产品架构

### 4.1 四大入口模块

| 模块 | 功能描述 |
|-----|---------|
| **个人主页** | 查看进度、推荐方向和快捷入口 |
| **职业咨询** | 真人前辈咨询 + 智能体咨询 |
| **职业分析** | 6步主流程（输入经历到行动指南） |
| **职场论坛** | 查看/发布职业经验帖子 |

### 4.2 职业分析主流程（6步）

1. **输入经历**：至少3段，每段包含情境/行动/结果/感受；支持示例一键填充
2. **偏好校准**：AI教练聊天追问（支持语音输入），提炼价值观与偏好
3. **职业分析选岗**：生成分层岗位推荐（含薪资区间），用户仅可选2个做对比
4. **职业体验模拟器**：选择确定方向后进入剧情模拟与评分反馈
5. **职业发展路径**：输出1年/3年/5年路径与风险提示
6. **行动指南**：给出30天验证任务与执行清单

## 5. 核心功能详解

### 5.1 成就经历采集

- 用户输入至少 3 段经历，每段包含：情境、行动、结果、感受
- 支持示例数据一键填充，降低首次使用门槛

### 5.2 职业画像生成（Career DNA）

输出三个维度：

- **能力簇（Skills）**：从经历中提炼的可迁移能力
- **驱动力（Drives）**：深层职业动机与价值取向
- **边界/不适配提醒（Boundaries）**：需要规避的工作类型

### 5.3 分层岗位推荐

- **第一梯队**：现在可试（高匹配度）
- **第二梯队**：可迁移（能力可迁移）
- **第三梯队**：探索潜力（长期发展）

每个岗位展示：匹配分、发展性、压力指数、标签

### 5.4 双岗位对比分析图谱

- 点击“开始对比”进入独立对比页
- 使用知识图谱结构展示影响因素节点（15个+）
- 节点支持动态弹性运动与手动拖拽
- 点击节点后展示 A/B 影响分析

**颜色语义：**
- **A 选项：橙色**
- **B 选项：蓝色**
- 节点按优势方自动着色

### 5.5 优势判断机制（Agent + Fallback）

- 优先尝试 Agent 联网分析各因子优势
- 若连接失败，自动切换为预设模拟优势数据
- 模拟数据中已预置各节点 A/B 优势与理由，保证结果可解释、可用

### 5.6 职业体验模拟器

- 支持剧情节点、随机事件、技能卡、连击与总分
- 提供过程反馈和最终评级
- 支持从岗位对比中锁定 A/B 后再进入模拟器
- 输出模拟结论、可能卡点与下一步验证建议

### 5.7 职业咨询（真人 + 智能体）

- 每位前辈提供真人咨询价格
- 每位前辈对应一个智能体咨询入口（定价约为真人 1/3）
- 点击“智能体咨询”进入独立聊天页
- 聊天页显示“该智能体由该前辈经历锻炼而成，并采用其语气”

### 5.8 职业发展路径分析

- 提供 Top 3 方向路径（1 年 / 3 年 / 5 年）
- 每条路径附风险提醒与验证任务

### 5.9 30 天行动指南

- 给出通用执行清单
- 针对已选意向岗位给出专项任务

---

## 6. 关键交互规则

- 第2步（偏好校准）不依赖已生成分析结果，可直接完成并进入下一步
- 第3步只允许选择 **2 个** 意向岗位进行对比
- 点击“选择A/选择B”仅锁定方向，不自动跳转；需用户点击“下一步”
- 所有主要状态保存在 `localStorage`，刷新后可恢复

- **数据持久化**：`localStorage`

- **AI能力**：
  - 偏好校准对话
  - 前辈智能体咨询
  - 对比因子联网判断（失败时回退模拟数据）

## 9. 本地运行方式

```bash
npm install
```

---

## 10. 当前限制

- 联网分析能力依赖外部模型服务与网络可用性
- 对比图谱中的“模拟优势”属于产品策略数据，不代表绝对客观结论
- 岗位与分析逻辑以规则引擎为主，非真实市场实时数据
- 模拟器为职业情境近似，不等同于真实企业岗位流程
