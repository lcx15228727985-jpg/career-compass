# Career Compass 职业指南针｜产品说明书

## 一、产品定位

**Career Compass 职业指南针**是一款面向在校生、应届生和早期职场人（0-5年）的职业方向决策产品。
核心不是"投哪份工作"，而是"先选对长期职业路径"。

产品通过用户过往成就事件与偏好对话，提炼职业画像（Career DNA），输出岗位推荐、双方案对比图谱和场景模拟体验，帮助用户做低成本但高质量的职业决策。

---

## 二、目标用户

| 用户群体 | 核心痛点 |
|---------|---------|
| 在校生/应届生 | 方向不明确，容易跟风选赛道 |
| 0-3年职场转型者 | 有经历但不知道哪个方向可持续 |
| 稳定性偏好者 | 不适合高强度竞争节奏，容易误判 |

---

## 三、核心价值与商业价值

### 3.1 核心价值主张

| 价值维度 | 具体说明 |
|---------|---------|
| **可解释决策** | 每个推荐与对比因子有理由，不是黑盒结论 |
| **先选路径，再做投递** | 从 Career Development 出发，而不是仅做岗位匹配 |
| **先模拟再投入** | 通过游戏化职业模拟器降低试错成本 |
| **人+AI混合咨询** | 既可约真人前辈，也可即时使用前辈智能体 |

### 3.2 商业价值

```
用户痛点                    解决方案                  商业价值
─────────────────────────────────────────────────────────────────
方向不清晰                 Career DNA 画像          降低决策焦虑
越努力越焦虑               反向筛选机制              减少无效投入
岗位推荐黑盒               可解释推荐+对比图谱       建立信任感
试错成本高                 职业体验模拟器            降低入行风险
真人咨询贵/难约             AI 智能体咨询             降低咨询门槛
```

**市场规模假设：**
- 每年应届毕业生约 1000 万+
- 0-3 年职场新人转换率约 30%
- 单用户付费意愿：99-399 元/次

### 5.2 核心 Agent 设计

#### Agent 1：偏好校准 Agent（Career DNA Builder）

```
功能：通过对用户成就经历的深度对话，提炼职业画像

工作流程：
┌────────────┐    ┌────────────┐    ┌────────────┐    ┌────────────┐
│  经历解析   │ → │  能力归因  │ → │  驱动力    │ → │  边界识别  │
│  Evidence  │    │  Skill    │    │  Drive    │    │  Boundary  │
│  Extraction│    │  Mapping  │    │  Mining   │    │  Detection │
└────────────┘    └────────────┘    └────────────┘    └────────────┘

输出：Career DNA 结构化画像
- 能力簇（Top 5-8）
- 驱动力/偏好（标签+解释）
- 边界/不适配因素（Top 3-5）
```

## 六、关键交互规则

- 第2步偏好校准不依赖已生成分析结果，可直接完成并进入下一步
- 第3步只允许选择2个意向岗位进行对比
- 点击"选择A/选择B"仅锁定方向，不自动跳转；需用户点击"下一步"
- 所有主要状态保存在 `localStorage`，刷新后可恢复
- 对比分析优先 Agent 联网，失败时自动回退模拟数据

## 九、当前限制

- 联网分析能力依赖外部模型服务与网络可用性
- 对比图谱中的"模拟优势"属于产品策略数据，不代表绝对客观结论
- 模拟器仍为职业情境近似，不等同真实组织环境
